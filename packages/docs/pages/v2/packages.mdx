# Packages

The source code of all packages locates [here](https://github.com/cosmology-tech/cosmos-kit)

Three workspaces:

- [packages](https://github.com/cosmology-tech/cosmos-kit/tree/main/packages) (base packages)
- [namespaces](https://github.com/cosmology-tech/cosmos-kit/tree/main/namespaces) (types of networks)
- [wallets](https://github.com/cosmology-tech/cosmos-kit/tree/main/wallets)

## Base Packages

### @cosmos-kit/core

- Exports commonly used `types`, `classes` and `utils`

### @cosmos-kit/react-lite

- Integrates React
- Exports `ChainProvider` and most `Hooks`

> - useChain
> - useChainWallet
> - useManager
> - useNameService
> - useWallet
> - useWalletClient

### @cosmos-kit/react

- Integrates React
- Provides default modal to select wallet
- Exports all from `@cosmos-kit/react-lite` and an additional `Hook`.

> - useModalTheme

### @cosmos-kit/walletconnect

- Exports `super classes` for wallets support [walletconnect v2](https://walletconnect.com/)
