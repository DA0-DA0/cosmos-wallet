# Sign

Basicall sign method is determined automatically by the doc type. But there are cases that multiple sign methods provided for one doc type, then parameter `options` is required to get more info.

## WalletConnect

If the doc is hex bytes string and namespace is `ethereum`, make sure `signHexBytes` is set.

**sign options**

```ts
{
    ethereum: {
        signHexBytes: 'personal_sign', // or 'eth_sign', cannot be undefined or null
    }
}
```
